# 🏆 Torneio Trijogo

Este projeto foi desenvolvido a fim aplicar os conhecimentos assimilados nas aulas da disciplina de Programação, além de incetivar o trabalho em equipe, o entretenimento e pensamento estratégico dos alunos.

A aplicação foi construída utilizando Linguagem C, permitindo os estudantes à aprenderem mais sobre essa liguagem de programação.

## 🎯 Objetivos

1. Tornar a aprendizagem da linguagem C mais lúdica para os pupilos.
2. Praticar os conhecimentos absorvidos em sala de aula. 
3. Compreender a lógica por trás da programação.
   

## Game Loop:

  A contrução do código do Torneio Trijogo foi estruturada em 5 partes

1.  Menu do Torneio.
2.  Perguntas e respostas.
3.  Cobra na caixa!
4.  Gousmans War.
5.  Teste de Código.
